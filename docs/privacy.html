OnTect 개인정보처리방침

시행일: 2026년 6월 12일

OnTect 무료 로컬 빌드는 학교 과학실의 시약품, 소모품, 기자재를 관리하기 위한 Android 앱입니다. 현재 무료 빌드는 네트워크 연결을 요구하지 않으며, 앱에 입력된 계정과 재고 데이터를 외부 서버로 전송하지 않습니다.

1. 저장되는 정보

OnTect 무료 로컬 빌드는 사용자가 앱 안에서 입력한 다음 정보를 Android 기기에 저장할 수 있습니다.

  • 로컬 계정의 이름, 이메일, 권한, 담당 학교 또는 실험실 이름
  • 로컬 로그인용 솔트 처리된 비밀번호 해시
  • 시약품, 소모품, 기자재의 이름, 분류, 수량, 보관 위치, 담당자, 위험도, 유통기한, 메모
  • 입고, 사용, 점검 기록
  • 앱 안에서 계산되는 관리자 통계

2. 정보 전송 및 공유

현재 무료 로컬 빌드는 저장된 데이터를 외부 서버, 광고 네트워크, 분석 서비스, 인증 제공자 또는 제3자에게 전송하지 않습니다.

3. 네트워크 사용

현재 무료 로컬 빌드는 Android 네트워크 권한을 요구하지 않습니다. 서버 동기화, 광고, 인앱 결제, 외부 로그인, 분석 SDK도 포함하지 않습니다.

4. 데이터 삭제

사용자는 Android 시스템 설정에서 OnTect 앱 저장공간을 삭제하거나 앱을 제거하여 로컬 계정과 재고 데이터를 삭제할 수 있습니다. 무료 로컬 빌드는 기기 안에 데이터를 저장하므로, 앱 저장공간을 삭제하면 해당 기기의 OnTect 데이터도 함께 삭제됩니다.

5. 학생 개인정보

OnTect는 학교 직원, 교사, 실험실 관리자가 재고를 관리하기 위한 도구입니다. 학생 개인정보 수집을 목적으로 설계되지 않았습니다.

6. 향후 변경

향후 클라우드 동기화, 외부 인증, Google Sheets 연동, 분석 기능 등이 추가되는 경우 앱 출시 또는 업데이트 전에 본 개인정보처리방침을 갱신합니다.

7. 문의

문의 또는 삭제 요청은 OnTect GitHub Issues 를 통해 접수할 수 있습니다.
