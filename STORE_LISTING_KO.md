# OnTect Google Play Listing Draft

## App Name

OnTect

## Short Description

학교 과학실 시약품, 소모품, 기자재를 기기 안에서 간편하게 관리하세요.

## Full Description

OnTect는 학교 과학실과 실험 준비실을 위한 무료 로컬 재고 관리 앱입니다.
스프레드시트만으로 관리하기 어려운 시약, 소모품, 기자재의 수량과 위치, 담당자, 유통기한, 입출고 기록을 한 화면에서 확인할 수 있도록 설계했습니다.

주요 기능:

- 첫 실행 관리자 계정 생성
- 교사, 실험 보조, 관리자 계정 추가
- 시약, 소모품, 기자재 등록 및 수정
- 보관 위치, 담당자, 위험도, 유통기한 기록
- 최소 재고 이하 항목 자동 표시
- 만료 임박 및 고위험 항목 확인
- 입고와 사용 기록 저장
- 관리자 대시보드에서 전체 사용자와 재고 통계 확인
- 서버, 광고, 유료 API 없이 기기 내 로컬 저장

무료 로컬 빌드 안내:

OnTect 무료 빌드는 현재 데이터를 외부 서버로 전송하지 않습니다.
계정과 재고 정보는 사용자의 Android 기기에 저장됩니다.
여러 기기에서 동시에 같은 데이터를 사용해야 하는 학교는 추후 학교 서버 또는 클라우드 동기화 버전이 필요할 수 있습니다.
