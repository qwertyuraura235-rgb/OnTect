# Privacy Policy

OnTect Free Build stores data locally on the Android device.

## Data Collected

The free build can store:

- demo login selection
- lab inventory items
- stock-in, stock-out, and audit records
- local admin dashboard statistics derived from inventory and demo users

## Data Sharing

The current free build does not send data to an external server.

## Network Use

The current free build does not require network access.

## Data Deletion

Users can remove local app data from Android system settings by clearing OnTect app storage or uninstalling the app.

## Future Changes

If cloud sync, external authentication, or Google Sheets integration is added later, this policy must be updated before release.
