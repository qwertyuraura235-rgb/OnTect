# OnTect Privacy Policy

Effective date: 2026-06-12

OnTect Free Build stores data locally on the Android device.
It does not require a network connection and does not send school inventory data to an external server.

## Data Collected

The free build can store:

- local account names, emails, roles, and lab names
- salted password hashes for local login
- lab inventory items
- stock-in, stock-out, and audit records
- local admin dashboard statistics derived from inventory and local users

## Data Sharing

The current free build does not send data to an external server.
It does not include ads, analytics SDKs, in-app purchases, Firebase, Supabase, or third-party login providers.

## Network Use

The current free build does not require network access.

## Data Deletion

Users can remove local app data from Android system settings by clearing OnTect app storage or uninstalling the app.
Because the free build is local-only, clearing app storage also removes local accounts and inventory records.

## Children and Schools

OnTect is intended as an inventory tool for school staff and lab managers.
It is not designed to collect personal information from students.

## Future Changes

If cloud sync, external authentication, or Google Sheets integration is added later, this policy must be updated before release.
